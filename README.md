## Simple note

This application is open source feel free to create pull requests, to rate the code, report bugs or suggest new features.

We do not collect any data.

Resources used

  * localization    
  * flutter_modular    
  * sqflite    
  * intl    
  * flutter_launcher_icons

Features

  * Create note
  * Update note
  * Delete note
  * Search note
  * Copy full note
